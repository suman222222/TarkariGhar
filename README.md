# TarkariGhar

TarkariGhar is a simple vegetable e-commerce platform designed to help users decide what to cook and purchase the required fresh ingredients easily.

## Project Overview

Most online grocery stores ask customers what vegetables they want to buy.

TarkariGhar approaches the problem differently by helping users choose a meal first and then providing the ingredients needed to prepare it.

The goal is to simplify meal planning, reduce shopping time, and make fresh vegetable purchasing more convenient.

## Current Features

* Homepage
* Navigation Bar
* Hero Section
* Meal Kit Display
* Meal Images
* Basic Website Structure

## Available Meal Kits

* Vegetable Chowmein
* Aloo Cauli
* Pumpkin Curry

Additional meal kits will be added in future versions.
